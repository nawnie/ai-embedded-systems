# Ai Embedded Systems

Ai Embedded Systems builds Rnv1 robotics and embedded AI systems for physical-world intelligence.

Website: https://aiembeddedsystems.com/

Current Cloudflare Pages fallback: https://ai-embedded-systems.pages.dev/

## Focus

- Rnv1 robotics prototype platform
- Embedded AI systems
- Sensor-driven perception
- Local onboard AI
- Sensor fusion, SLAM, and detection
- Memory and data for robot experience
- Robust real-world control

## Website Source

This repository contains the public Vite/React website source for Ai Embedded Systems.

```powershell
npm install
npm run build
```

The production build output is `dist/`.

Public crawler files live in `public/`:

- `robots.txt`
- `sitemap.xml`
- `llms.txt`
- `AGENTS.md`
- `humans.txt`

The canonical site target is `https://aiembeddedsystems.com/`. The earlier hyphenated domain should redirect to the canonical no-hyphen domain after Cloudflare DNS migration is complete.
